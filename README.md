# Weather App

A modern, responsive weather application that displays current conditions and 7-day forecasts using the Open-Meteo API.

## Features

- **Current Weather**: Temperature, humidity, wind speed, pressure, and feels-like temperature
- **7-Day Forecast**: Daily high/low temperatures, weather conditions, and precipitation probability
- **Interactive Charts**: 24-hour temperature and wind speed trends, 7-day precipitation chart (Chart.js)
- **Dual Input Modes**: Search by city name or enter coordinates directly
- **Unit Toggle**: Switch between Celsius/Fahrenheit with persistent preference
- **Auto Unit Detection**: Defaults to Fahrenheit for US locations, Celsius elsewhere
- **Responsive Design**: Works on desktop, tablet, and mobile

## APIs Used

- [Open-Meteo](https://open-meteo.com/) - Weather data
- [Nominatim (OpenStreetMap)](https://nominatim.openstreetmap.org/) - Geocoding and reverse geocoding

## Usage

Open `index.html` in a browser. No build step required.

## Tech Stack

- HTML5 / CSS3 / Vanilla JavaScript
- Chart.js (CDN)
